# Swedish Employee Cost Calculator

En webapp för att beräkna den totala kostnaden för att ha en anställd i Sverige.

## Funktioner

- Beräkning av bruttolön till nettolön
- Skattetabeller baserat på ålder och kommun
- Sociala avgifter för arbetsgivare
- Kyrkoavgift
- Pensionsavgifter
- Försäkringar

## Teknisk stack

- Next.js med TypeScript
- Tailwind CSS för styling
- React för frontend

## Installation

```bash
npm install
npm run dev
```
