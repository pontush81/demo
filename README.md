# Swedish Employee Cost Calculator

En webapp för att beräkna den totala kostnaden för att ha en anställd i Sverige.

## Funktioner

- Beräkning av bruttolön till nettolön
- Skattetabeller baserat på ålder och kommun (alla 290 svenska kommuner)
- Sökbar kommunlista för enkel navigation
- Sociala avgifter för arbetsgivare (31,42%)
- Kyrkoavgift (1,04% för medlemmar)
- Pensionsavgifter och försäkringar
- Åldersbaserade variationer i skattesatser

## Teknisk stack

- React med Vite
- TypeScript
- Tailwind CSS för styling
- shadcn/ui komponenter
- Lucide React ikoner

## Installation

```bash
npm install
npm run dev
```
